# 収支トラッカーアプリ
収支を記録できるウェブアプリケーションです。<br>
フロントエンドはReact+JavaScript<br>
バックエンドはFastAPI

# 機能概要
* 収入、支出の切り替え
* 日付選択可能
* 登録した情報の削除
* 収支の残高表示
